Honky Tonkify

This was my midstone project at Nashville Software School. It was built out using React JS and json-server for the data pulled. There is a random lyric generator I built in that will allow you to randomly generate lyrics based on a simplistic model of a sentence. It fits them together in a way that while linguistically makes sense--doesn't quite semantically. This was part of the fun as I realized what a complicated problem this was, but also how humorous it was to have the computer trying to formulate meaningful sentences.
